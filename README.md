## 📋 About

FIT-BULL is a gym and fitness club landing page designed to attract new members. It features a powerful hero section, trainer showcases, workout program highlights, and a membership CTA — all styled with a bold, high-energy aesthetic.

---

## ✨ Features

- **Hero section** with full-screen background image and headline CTA
- **Trainer profiles** — showcasing male and female fitness coaches
- **Workout programs** section with program cards
- **Membership plans** with pricing highlights
- **Responsive layout** built with CSS Flexbox
- **Multi-image gallery** with background sections
